NYCICC Cyber Ministry — 2025

Worship Lyrics Workflow

  • Lock song order ≥48h before (coordinate with worship leader)
  • Pull lyrics from the official Song Book; verify alignment
  • Proofread line breaks, capitalization, readability
  • Accessibility: ≥48pt font, high contrast, no busy backgrounds
  • Rehearsal run-through; export backup PDF/PNG to USB

Compliance: ensure licensing (CCLI if displayed); avoid posting full lyrics publicly

NYCICC Cyber Ministry — 2025

Livestream Prep (YouTube + OBS)

  • Schedule YouTube livestream ≥1 day before
  • Set title, description, tags, thumbnail, category
  • OBS scenes: Pre, Welcome, Sermon, Lyrics/L3, Offering, Communion, Outro
  • Label sources; optional hotkeys; prebuilt lower-thirds
  • Run a 2-minute unlisted test; record locally; auto-start/stop off
  • Audio: target LUFS -18 to -14; peaks around -6 dB
NYCICC Cyber Ministry — 2025

Audio/Video Routing

  • Cameras: connect/label; set white balance and exposure; match frame rates
  • Option 1: DJI Mic → laptop; OBS source = “Mic Receiver”
  • Option 2: Sound board monitor out → camera → USB capture; OBS audio = “Camera”
  • Monitor with closed-back headphones; 80 Hz HPF on speech; minimal AGC
  • Keep peaks ~ -6 dB; maintain voice clarity over music
  • Redundancy: spare cables, batteries, backup mic/transmitter
NYCICC Cyber Ministry — 2025

Network & Fail-safes

  • Upload ≥10 Mbps; test 30 min pre-service (speed + ping)
  • Hotspot/secondary connection ready
  • UPS on mixer, streaming PC, router/ONT
  • Save OBS profiles/scenes to cloud + USB
  • If stream drops: keep recording; post replay within 2 hours
  • Log incidents: issue, time, fix, owner, prevention

Quality: hook in 2s · captions on all videos · strong thumbnail contrast

Appendix — Quick Checklists

Pre-Service (Tech)

  • Upload ≥10 Mbps; hotspot ready
  • Cameras connected; batteries/SD checked
  • Audio route set (Mic Receiver or Camera)
  • OBS scenes verified; record enabled; test done
  • YouTube stream scheduled ≥1 day before

Pre-Service (Lyrics)

  • Song order verified (≥48h lock)
  • Lyrics from official Song Book; formatting checked
  • Backup slides to USB; accessibility confirmed

Post-Service

  • Stop stream/record; verify local file
  • Upload sermon/clip assets
  • Log incidents and action items
  • Begin content clipping (60–90s highlight)
